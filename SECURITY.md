# Security Policy

## Supported Versions
This security policy applies to all versions of the **PC Hotspot Manager** scripts (`run v1.bat`).

---

## Reporting a Vulnerability
If you discover a security issue in this project, please report it responsibly:

- Send an email to: **feljemenor236@gmail.com**
- Include:
  - Steps to reproduce the issue
  - Expected vs actual behavior
  - Any screenshots or error logs

---

## Security Guidelines
- Always **run scripts as Administrator** to work properly.
- Do **not share these scripts** with untrusted users.
- Only use the scripts on computers you own or are authorized to manage.
- Do **not modify system files** outside of what the scripts are intended to change.

---

## Handling Sensitive Data
This project does **not collect or store any personal or network data**. It only modifies system settings related to **PC Mobile Hotspot**.
